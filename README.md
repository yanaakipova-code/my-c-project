## building_generator.c - почему именно такие вероятности:

### Параметры генерации зданий:

- `#define DEVELOPER_MICRODISTRICT_COUNT 6` - Количество застройщиков
- `#define MICRODISTRICT_COUNT 7` - Количество микрорайонов  
- `#define BUILDING_TYPE_COUNT 3` - Типы строительства: панель, кирпич, монолит

### Года постройки домов:

- **1970** - откуда начали строить
- **25 лет** - охватывает советский период строительства
- **2000** - 'граница' между старыми и современными домами

### Чисто из логики( нельзч остасить многоэтажный дом без лифта ):

- `#define MIN_FLOORS 1` - Минимум этажей (допустим для магазина или пристройки)
- `#define MAX_FLOORS 28` - Максимальная этажность в типовой застройке
- `#define MANDATORY_ELEVATOR_FLOORS 9` - Лифт обязателен с 10 этажа
- `#define MIN_APARTMENTS_PER_FLOOR 4` - Минимальное количество квартир
- `#define MAX_APARTMENTS_PER_FLOOR 12` - Максимальное количество квартир

## vector.h - о структуре:
### Структура вектора:
- data - указатель на динамический массив элементов
- size - Сколько сейчас элементов в векторе 
- capacity - Сколько вектор может в себя вместить элементов

## vector.h - какие есть функции:
- vector_size - размер вектора( возвращает текущее кол-во элементов в векторе )
- Функция vector_get - безопасный доступ к структуре :
- vector_is_valid - вектор создан и выделена память
- vector_is_valid_and_not_empty - в векторе есть данные и если он создан
- vector_front - доступ к первому элементу структуры
- vector_back - доступ к последнему элементу структуры
- vector_expansion - увеличение памяти 
- vector_push_back - добавка в конец
- vector_push_front - добавка в начало
- vector_insertion - добавка в конкретное место
## args.c - описание функций
### parse_arguments - функция "перевода"
**Параметры:**
- **`int argc`** - количество аргументов командной строки
- **`char* argv[]`** - массив строк аргументов
- **`ProgramArgs* args`** - структура для сохранения распарсенных значений 
**Значения по умолчанию:**
- **`args->mode = MODE_ERROR`** - режим пока не указан
- **`args->order = ORDER_ASC`** - сортировка по возрастанию (по умолчанию)
- **`args->input_file = NULL`** - входной файл не указан (чтение с клавиатуры)
- **`args->output_file = NULL`** - выходной файл не указан (вывод на экран)
- **`args->generate_count = 0`** - количество генерируемых записей (0 = не генерировать)
