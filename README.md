# Моя третья лаба

## building_generator.c - почему именно такие вероятности:

### Параметры генерации зданий:

- `#define DEVELOPER_MICRODISTRICT_COUNT 6` - Количество застройщиков
- `#define MICRODISTRICT_COUNT 7` - Количество микрорайонов  
- `#define BUILDING_TYPE_COUNT 3` - Типы строительства: панель, кирпич, монолит

### Года постройки домов:

- **1970** - Начало эры массового индустриального домостроения
- **25 лет** - охватывает советский период строительства
- **2000** - 'граница' между старыми и современными домами

### Чисто из логики( нельзч остасить многоэтажный дом без лифта ):

- `#define MIN_FLOORS 1` - Минимальная этажность (редко, но бывает)
- `#define MAX_FLOORS 28` - Максимальная этажность в типовой застройке
- `#define MANDATORY_ELEVATOR_FLOORS 9` - Лифт обязателен с 10 этажа
- `#define MIN_APARTMENTS_PER_FLOOR 4` - Минимальное количество квартир
- `#define MAX_APARTMENTS_PER_FLOOR 12` - Максимальное количество квартир
